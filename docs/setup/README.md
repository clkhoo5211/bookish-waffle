# Setup & Configuration

Configuration guides for setting up the RVMplus Dapps project.

## Files

- **GITHUB_SETUP.md** - GitHub repository and deployment setup
- **PRIVY_SETUP.md** - Privy authentication configuration
- **TOKEN_ADDRESSES.md** - Smart contract addresses and environment variables
- **SETUP_COMPLETE.md** - Setup completion checklist

## Quick Start

1. Clone the repository (see GITHUB_SETUP.md)
2. Install dependencies: `npm install`
3. Configure Privy (see PRIVY_SETUP.md)
4. Set environment variables (see TOKEN_ADDRESSES.md)
5. Run development server: `npm run dev`
