## üé® RESPONSIVE DESIGN PATTERNS USED

### 1. **Mobile-First Approach**
```css
/* Default (Mobile) */
text-sm p-3 gap-1

/* Tablet/Desktop Override */
md:text-base md:p-4 md:gap-2
```

### 2. **Container Constraints**
```css
/* Mobile: Narrow for readability */
max-w-[480px]

/* Tablet: Medium width */
md:max-w-2xl

/* Desktop: Wide but still constrained */
lg:max-w-4xl
```

### 3. **Icon Scaling**
```css
/* Mobile: Small */
size={24} w-10 h-10

/* Desktop: Larger */
md:w-7 md:h-7 md:w-12 md:h-12
```

### 4. **Typography Scaling**
```css
/* Balance Display */
text-3xl md:text-4xl lg:text-5xl

/* Body Text */
text-sm md:text-base
```

### 5. **Spacing Adjustments**
```css
/* Gaps */
gap-1 md:gap-2

/* Margins */
mb-6 md:mb-8

/* Padding */
p-3 md:p-4
```
