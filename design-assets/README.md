# Design Assets

This directory contains all design assets and mockups for the RVM Web3 Payment PWA project.

## Directory Structure

```
design-assets/
├── figma-mockups/          # Original Figma design mockups
├── wireframes/             # Wireframes (to be created by UX Agent)
├── design-system/          # Design system documentation (to be created)
└── README.md              # This file
```

## Figma Mockups

The `figma-mockups/` directory contains the original design mockups from the RVM design2 directory. These serve as the reference for UI/UX implementation.
