### Basic Information
- **Name**: Alex Chen

- **Location**: San Francisco, CA
- **Occupation**: DeFi Developer
- **Tech Savviness**: Very High

### Demographics
- **Education**: Bachelor's in Computer Science
- **Income**: $120K+
- **Crypto Experience**: 5+ years

### Background
Alex is an experienced crypto user who actively participates in DeFi protocols, NFT marketplaces, and Web3 projects. They hold assets across multiple blockchain networks (Ethereum, Polygon, Arbitrum, Optimism) and frequently makes crypto payments for services, products, and transactions.

### Goals & Motivations
- **Primary Goal**: Efficiently manage multi-chain payments without switching wallets
- **Secondary Goal**: Track payments and transactions across all chains
- **Motivation**: Save time and reduce complexity in multi-chain operations

### Pain Points
- **Frustration**: Switching between multiple wallets for different chains
- **Challenge**: Managing transaction history across chains
- **Problem**: Complex payment flows for simple transactions
- **Need**: Unified interface for all crypto payment needs

### Behaviors
- Uses MetaMask, WalletConnect, and other wallets
- Makes 10-20 crypto transactions per month
- Actively participates in DeFi and NFT communities
- Prefers self-custody and control over assets
- Values security and transparency

### Technology Preferences
- **Wallets**: MetaMask, WalletConnect, hardware wallets
- **Chains**: Ethereum, Polygon, Arbitrum, Optimism, Base
- **Devices**: Desktop browser, mobile wallet apps
- **Preferences**: Open-source, decentralized, non-custodial

### Product Expectations
- **Must Have**: Multi-chain support, MetaMask integration, transaction history
- **Should Have**: Gas optimization, advanced analytics, API access
- **Nice to Have**: Batch transactions, gas price alerts, custom token lists

### Success Metrics
- **Adoption**: Uses app for 80%+ of crypto payments
- **Retention**: Daily active user, 90%+ monthly retention
- **Value**: Processes $5K+ in transactions per month

### Quote
*"I just want to pay with crypto without thinking about which chain I'm on. A unified interface that works across all my wallets would be perfect."*

- **Name**: Sarah Martinez
- **Age**: 28
- **Location**: Austin, TX
- **Occupation**: Marketing Manager
- **Tech Savviness**: Medium

### Demographics
- **Education**: Bachelor's in Marketing
- **Income**: $75K
- **Crypto Experience**: < 1 year
- **Device Usage**: Mobile (80%), Desktop (20%)

### Background
Sarah is new to crypto but interested in using it for payments. She's heard about crypto payments but finds the setup process intimidating. She wants a simple, mobile-friendly way to start using crypto without complex wallet setups or technical knowledge.

### Goals & Motivations
- **Primary Goal**: Start using crypto payments easily and safely
- **Secondary Goal**: Learn about crypto payments gradually
- **Motivation**: Join the crypto economy without technical complexity

### Pain Points
- **Frustration**: Complex wallet setup and seed phrase management
- **Challenge**: Understanding gas fees and transaction costs
- **Problem**: Fear of making mistakes with crypto
- **Need**: Simple, guided experience with embedded wallets

### Behaviors
- Interested in crypto but hesitant to start
- Prefers mobile-first experiences
- Values simplicity and ease of use
- Needs guidance and clear instructions
- Wants to start small and learn gradually

### Technology Preferences
- **Wallets**: Embedded wallets, social login
- **Chains**: Ethereum, Polygon (simple, popular chains)
- **Devices**: Mobile phone (primary), tablet
- **Preferences**: Simple, intuitive, safe, guided

### Product Expectations
- **Must Have**: Embedded wallet, social login, simple UI, clear instructions
- **Should Have**: Educational content, transaction guides, customer support
- **Nice to Have**: Tutorials, tips, best practices

### Success Metrics
- **Adoption**: Creates wallet and makes first transaction
- **Retention**: Weekly active user, 50%+ monthly retention
- **Value**: Processes $100-500 in transactions per month

### Quote
*"I want to use crypto payments, but I'm worried about setting up a wallet wrong. If I could just sign in with my Google account and start paying, that would be perfect."*
